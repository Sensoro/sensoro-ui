所有的 daisyUI 组件

55 组件

Accordion 手风琴

手风琴用于显示和隐藏内容，但一次只能打开一个项目。

Alert 警告提示

警告提示用来提示用户重要的事件消息。

Artboard 画板

Artboard 提供了固定尺寸的容器，用于在手机尺寸里显示演示内容。

Avatar 头像

头像用于在界面中显示个人或企业的缩略图。

Badge 徽章

徽章是用来通知用户特定数据的状态。

Bottom navigation 底部导航栏

底部导航栏允许在主屏幕之间导航。

Breadcrumbs 面包屑导航

面包屑导航可以帮助用户导航网站。

Button 按钮

按钮可以让用户执行命令或者作出选择。

Card 卡片

卡片用于显示易于阅读的显示内容。

Carousel 轮播

轮播可以在可滚动区域里显示图像或内容。

Chat bubble 聊天气泡

聊天气泡用于显示一行对话及其所有数据，包括作者图像、作者名称、时间等。

Checkbox 复选框

复选框用于选择或取消选择值。

Collapse 折叠面板

折叠面板用于显示和隐藏内容。

Countdown 计数

计数给了你改变数字的过渡效果。

Diff 对比

Diff 组件显示两个项目的并排比较。

Divider 分隔线

分隔线用于垂直或水平分隔内容。

Drawer 抽屉

抽屉是一种可以显示/隐藏页面左侧或右侧的侧边栏的网格布局。

Dropdown 下拉菜单

下拉菜单可以在点击按钮时打开菜单或其他元素。

文件输入框

文件输入框是一个用于上传文件的输入框。

Footer 页脚

页脚可以包括标志、版权声明和链接到其他网页。

Hero 故事

Hero 是显示带有标题和描述的大方框或图像的组件。

Indicator 标志

标志用来将元素放置在另一个元素的角落。

文字输入框

文字输入框是简单的输入框。

Join 组合

Join 是用于对多个项目进行分组的容器，它可用于对按钮、输入或任何其他元素进行分组。Join 将边框半径应用于第一个和最后一个项目。Join 可用于创建一个水平或垂直的项目列表。

Kbd 键盘

键盘用于显示键盘快捷键的效果。

Link 链接

链接将缺失的下划线样式添加到链接中。

Loading 加载

加载会显示动画以指示正在加载某些内容。

Mask 遮罩

Mask 将元素的内容修剪成常见的形状。

Menu 菜单

菜单用于垂直或水平显示链接列表。

浏览器模型

浏览器模型展示了一个看起来像浏览器窗口的框。

代码模型

代码模型用于在看起来像代码编辑器的方框中显示代码块。

手机模型

手机模型显示了一个像 iPhone 的模型。

视窗模型

视窗模型展示了一个看起来像操作系统窗口的框。

Modal 对话框

对话框用于点击按钮时显示对话框或一个盒子模型。

Navbar 导航栏

导航栏用于在页面顶部显示导航栏。

Pagination 分页

分页是一组按钮，允许用户在一系列相关内容之间移动。

Progress 进度条

进度条可以用来显示任务的进展情况，也可以用来显示时间的流逝。

Radial progress 圆形进度条

圆形进度条可以用来显示任务的进度，也可以用来显示时间的流逝。

Radio 单选框

使用单选框，用户可以从多个选项中选择一个选项。

范围滑块

范围滑块用于滑动选择值。

Rating 评分

评价是一组单选按钮，用户可以对某件事进行评价。

Select 选择器

选择器用于从选项列表中选择值。

Skeleton 骨架屏

Skeleton 是一个可以用来显示组件加载状态的组件。

Stack 堆叠

堆叠使元素在视觉上有重叠的效果。

Stat 统计展示

统计展示用于在方框中显示数值和数据。

Steps 步骤条

你可以使用步骤条来显示流程步骤的列表。

Swap 调换

使用 Swap，你可以使用复选框或类名来切换两个元素的显示。

Tabs

你可以使用选项卡，以选项卡的形式显示链接列表。

Table 表格

你可以使用表格来显示数据列表。

Textarea 多行文本框

多行文本框允许用户在多行中输入文本。

主题控制器

如果页面中存在带有 theme-controller 类的选中复选框输入或选中的单选框输入，则页面将具有与该输入值相同的主题。

Timeline 时间轴

Timeline 组件按时间顺序显示事件列表。

Toast 弹框

Toast 是一个用于堆叠元素的包装器，位于页面的角落。

Toggle 开关

开关是为了看起来像开关按钮的样式的复选框。

Tooltip 文字提示

你可以使用文字提示，当用户使用点击或者鼠标对准某个元素时显示消息。
